ការអនុវត្តគម្រោងជាក់ស្តែង

តើ Smart Devices ជាអ្វី?

Smart Device គឺជាឧបករណ៍អេឡិចត្រូនិចដែលអាចធ្វើការគិតវិនិច្ឆ័យ និងឆ្លើយតបដោយស្វ័យប្រវត្តិ។

វាដំណើរការដោយការរួមបញ្ចូលគ្នា៖

  • មានខួរក្បាល (Computing)
  • មានវិញ្ញាណ (Sensing)
  • អាចតភ្ជាប់ (Connectivity)

រូបមន្តគ្រឹះនៃឧបករណ៍ឆ្លាតវៃ

Smart Device = (Computing + Sensing) × Connectivity

"ឧបករណ៍មួយអាចចាត់ទុកជា Smart Device បាន ក៏ដោយសារវាមានសមត្ថភាព គិត, ដឹង, និង ទំនាក់ទំនង។"

១. ខួរក្បាល (Computing Unit)

Microcontroller / CPU

ជាផ្នែកដែលទទួលខុសត្រូវលើការគណនា និងវិភាគទិន្នន័យដែលទទួលបានពី Sensor ដើម្បីធ្វើការសម្រេចចិត្ត។

ឧទាហរណ៍៖ Arduino, ESP32, Raspberry Pi, Micro-Chips...

២. វិញ្ញាណ (Sensing Unit)

Temp/Humidity

Light (LDR)

Motion (PIR)

Ultrasonic

Sensors (វិញ្ញាណ)

Sensors គឺជាឧបករណ៍ដែលជួយឱ្យ Smart Device អាច "ដឹង" ពីអ្វីដែលកើតឡើងនៅបរិស្ថានជុំវិញ។

វាបំប្លែងបាតុភូតរូបវិទ្យា (កម្តៅ, ពន្លឺ, សំឡេង) ទៅជាសញ្ញាអេឡិចត្រូនិច។

៣. សកម្មភាព (Action Unit)

Actuators

Motor, Servo, Relay, Valve

Outputs

LED, LCD, OLED, Buzzer

តួនាទី

បំប្លែងបញ្ជាអេឡិចត្រូនិចទៅជាសកម្មភាពពិត

"ខួរក្បាលបញ្ជា វិញ្ញាណផ្តល់ដំណឹង ហើយសកម្មភាពជាអ្នកប្រតិបត្តិ។"

៤. ការតភ្ជាប់ (Connectivity)

Internet of Things (IoT)

ជាផ្នែកដែលធ្វើឱ្យឧបករណ៍អាច "និយាយគ្នា" ឬបញ្ជាពីចម្ងាយបាន។

  • Wi-Fi (ESP32, NodeMCU)
  • Bluetooth (HC-05, BLE)
  • Radio / LoRa / NB-IoT

Language: C++ Based
Environment: Arduino IDE

១. pinMode(pin, mode)

មុខងារ៖

ប្រើសម្រាប់កំណត់មុខងារ Pin នីមួយៗមុននឹងប្រើប្រាស់។

  • OUTPUT: បញ្ចេញសញ្ញា (LED, Motor)
  • INPUT: ទទួលសញ្ញា (Sensor, Button)
void setup() {
  // កំណត់ Pin 13 ជាអ្នកបញ្ចេញ
  pinMode(13, OUTPUT);

  // កំណត់ Pin 2 ជាអ្នកទទួល
  pinMode(2, INPUT);
}

២. digitalWrite(pin, value)

មុខងារ៖

បញ្ជូនតម្លៃ HIGH (5V) ឬ LOW (0V) ទៅកាន់ Digital Pin។

HIGH / ON
LOW / OFF
void loop() {
  digitalWrite(13, HIGH); // បើកភ្លើង
  delay(1000);
  digitalWrite(13, LOW);  // បិទភ្លើង
  delay(1000);
}

៣. digitalRead(pin)

int buttonState = digitalRead(2);

if (buttonState == HIGH) {
  // បើមានគេចុចប៊ូតុង
  digitalWrite(13, HIGH);
} else {
  digitalWrite(13, LOW);
}

មុខងារ៖

ប្រើសម្រាប់អានតម្លៃពី Digital Pin (ទទួលបានតម្លៃ 0 ឬ 1)។

Note:

ជាទូទៅប្រើសម្រាប់អានប៊ូតុង (Button) ឬ PIR Motion Sensor។

៤. analogRead(pin)

មុខងារ៖

អានតម្លៃអាណាឡូកពី Pin A0-A5។

0 - 1023

Resolution: 10-bit ADC

int value = analogRead(A0);

// បំប្លែងតម្លៃទៅជាភាគរយ (0-100%)
int percent = map(value, 0, 1023, 0, 100);

Serial.println(percent);
