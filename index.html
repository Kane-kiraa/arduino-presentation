តើ Smart Devices ជាអ្វី?

Smart Device គឺជាឧបករណ៍អេឡិចត្រូនិចដែលមានសមត្ថភាព គិត (Computing), ដឹង (Sensing) និង ធ្វើការឆ្លើយតបដោយស្វ័យប្រវត្តិ។

  • ផ្នែករឹង & កម្មវិធី (Hardware & Software): ប្រើប្រាស់ CPU ឬ Microcontroller ជាខួរក្បាល។
  • ការទទួល & ប្រតិបត្តិ (Input/Output): ប្រើប្រាស់ Sensor សម្រាប់ទទួលព័ត៌មាន និង Actuator សម្រាប់ធ្វើសកម្មភាព។
  • ការតភ្ជាប់ (Connectivity): អាចភ្ជាប់ទៅកាន់ Internet ឬ Network ដើម្បីផ្លាស់ប្តូរទិន្នន័យ។

រូបមន្តគ្រឹះនៃឧបករណ៍ឆ្លាតវៃ

Smart Device = (Computing + Sensing) × Connectivity

ការសរសេរកម្មវិធី Arduino

Functions & Methods សំខាន់ៗ

Structure

setup(), loop()

Digital I/O

pinMode(), digitalWrite(), digitalRead()

1. pinMode(pin, mode)

កំណត់មុខងារនៃ Pin នីមួយៗមុនពេលប្រើប្រាស់។

  • OUTPUT: កំណត់ជាច្រកបញ្ចេញ (LED, Relay)
  • INPUT: កំណត់ជាច្រកទទួល (Sensor, Button)
  • INPUT_PULLUP: ទទួលសញ្ញាជាមួយ Resistor ខាងក្នុង 
void setup() {
  pinMode(13, OUTPUT);
  pinMode(2, INPUT);
}

2. digitalWrite(pin, value)

បញ្ជាឱ្យ Pin បញ្ចេញតង់ស្យុង។

  • HIGH = 5V (ឬ 3.3V) => បើក (ON)
  • LOW = 0V (GND) => បិទ (OFF) 
void loop() {
  digitalWrite(13, HIGH); // Turn LED ON
  delay(1000);
  digitalWrite(13, LOW);  // Turn LED OFF
  delay(1000);
}

3. digitalRead(pin) 

int state = digitalRead(2);

if (state == HIGH) {
  // Button is pressed
  digitalWrite(13, HIGH);
} else {
  digitalWrite(13, LOW);
}

អានតម្លៃពី Digital Pin

លទ្ធផលទទួលបានមានតែ ២ គឺ៖

1 (HIGH)
0 (LOW)

4. analogRead(pin)

អានតម្លៃ Analog (ពីសេនស័រ)

0 - 1023

10-bit Resolution (Arduino Uno)

// Pin A0 connected to Sensor
int sensorValue = analogRead(A0);

// Convert to voltage (optional)
float voltage = sensorValue * (5.0 / 1023.0);

4. ការតភ្ជាប់ (Connectivity)

Internet of Things (IoT)

  • Wi-Fi (ESP32, ESP8266)
  • Bluetooth (HC-05, BLE)
  • LoRa / GSM

រចនាសម្ព័ន្ធកូដ Arduino 

void setup() {
  // កូដនេះដំណើរការតែម្ដងគត់ នៅពេលបើកភ្លើង
  pinMode(13, OUTPUT);
}

void loop() {
  // កូដនេះដំណើរការចុះឡើង ជារៀងរហូត
  digitalWrite(13, HIGH);
  delay(1000);
  digitalWrite(13, LOW);
  delay(1000);
}
